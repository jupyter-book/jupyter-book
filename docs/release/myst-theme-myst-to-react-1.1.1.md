This brings in several UI and UX improvements that were noticed after the last minor release!

## Bugs fixed

- Hover preview cards to documents with metadata now work! [#772](https://github.com/jupyter-book/myst-theme/pull/772) ([`@choldgraf`](https://github.com/choldgraf), [`@bsipocz`](https://github.com/bsipocz), [`@rowanc1`](https://github.com/rowanc1), [`@stefanv`](https://github.com/stefanv))
- Highlighting during scrolls and clicks in the "Contents" section is now fixed [#778](https://github.com/jupyter-book/myst-theme/pull/778) ([`@choldgraf`](https://github.com/choldgraf), [`@bsipocz`](https://github.com/bsipocz), [`@rowanc1`](https://github.com/rowanc1), [`@stefanv`](https://github.com/stefanv))
- BASE_URL for `{static}` downloads is now fixed, it was being added twice [#770](https://github.com/jupyter-book/myst-theme/pull/770) ([`@rowanc1`](https://github.com/rowanc1), [`@choldgraf`](https://github.com/choldgraf), [`@ryanlovett`](https://github.com/ryanlovett))
- Increase contrast of search bar on dark mode [#768](https://github.com/jupyter-book/myst-theme/pull/768) ([`@choldgraf`](https://github.com/choldgraf), [`@rowanc1`](https://github.com/rowanc1))
- Make contents tab have a background to hide margin content [#775](https://github.com/jupyter-book/myst-theme/pull/775) ([`@choldgraf`](https://github.com/choldgraf), [`@rowanc1`](https://github.com/rowanc1), [`@stefanv`](https://github.com/stefanv))
- Update and fix icon link class [#773](https://github.com/jupyter-book/myst-theme/pull/773) ([`@choldgraf`](https://github.com/choldgraf))

## Documentation improvements
